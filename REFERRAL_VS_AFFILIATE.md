# Referral Program vs Affiliate Program

## Key Differences

The platform now has TWO distinct earning programs that work independently:

---

## Referral Program
**Location:** Profile → Referral

### Purpose
Simple direct referral rewards for bringing new users to the platform.

### Structure
- **Single-tier only**: You earn from your direct referrals
- No sub-affiliate tracking
- No multi-level commission chains
- Simple invite-and-earn model

### Commission Rates
- 10-70% based on VIP level (same as Tier 1 of Affiliate Program)
- Direct earnings only
- No override commissions

### Features
- Share referral code
- View direct referrals
- Track personal earnings
- Monthly earnings display
- Volume tracking for VIP progression

### Best For
- Users who want simple, direct rewards
- Quick earnings from personal network
- No complex network building

---

## Affiliate Program
**Location:** Profile → Affiliate Program

### Purpose
Professional multi-level marketing system for serious network builders.

### Structure
- **5-tier deep**: Earn from referrals 5 levels down
- Complete network tree tracking
- Sub-affiliate commission chains
- Professional MLM structure

### Commission Rates
- **Tier 1**: 10-70% (same as referral program)
- **Tier 2**: 20% of Tier-1 commission
- **Tier 3**: 10% of Tier-1 commission
- **Tier 4**: 5% of Tier-1 commission
- **Tier 5**: 2% of Tier-1 commission

### Features
- Complete network visualization
- 5-tier breakdown statistics
- Total network size tracking
- Sub-affiliate management
- Earnings calculator
- Commission history per tier
- CPA bonus options
- Hybrid compensation plans

### Best For
- Professional affiliates
- Network marketers
- Users building large downlines
- Passive income seekers
- Long-term income generation

## Example Scenario

### Referral Program
You refer 10 people who each trade $10,000/month.

**Your earnings:**
- 10 × $10,000 × 0.1% fee × 50% commission = $500/month

### Affiliate Program
You refer 10 people who each refer 5 people who each refer 3 people (T1: 10, T2: 50, T3: 150).

**Your earnings if everyone trades $10,000/month:**
- Tier 1 (10 people): $500/month (same as referral)
- Tier 2 (50 people): $500/month (20% override)
- Tier 3 (150 people): $750/month (10% override)
- Tier 4 (450 people): $1,125/month (5% override)
- Tier 5 (1,350 people): $1,350/month (2% override)

**Total: $4,225/month** (8.45x more than referral-only)

## How They Work Together

### Same Referral Code
- Both systems use your referral code
- When someone signs up with your code, they count for both programs
- Earnings are separate and additive

### VIP Progression
- Volume from both systems counts toward VIP level
- Higher VIP = Higher commissions in both programs

### Wallet Credits
- Both programs credit USDT to your main wallet
- Separate transaction records
- Combined in total earnings display

## Which One Should You Use?

### Choose Referral Program If:
- You have a small personal network
- You prefer simplicity
- You don't want to build a team
- You just want direct rewards

### Choose Affiliate Program If:
- You want to build a large network
- You have team-building skills
- You want passive income
- You're serious about earning

### Use Both!
**Recommended approach:**
- You automatically participate in both
- Focus on building your network
- Affiliate program will generate more income
- Referral page provides simple overview
- Affiliate page provides detailed analytics

## Technical Implementation

### Database
- Referral data: `referral_stats`, `referral_commissions`
- Affiliate data: `affiliate_tiers`, `tier_commissions`
- Shared: `user_profiles.referred_by`, VIP levels

### Commission Distribution
- Both systems process on every trade
- Separate function calls
- No conflicts or double-counting
- Real-time calculation

### UI Access
- Referral: Simple overview page
- Affiliate: Comprehensive dashboard with 5 tabs

## Summary

**Referral Program** = Simple, direct, 1-tier earnings
**Affiliate Program** = Advanced, multi-tier, network building

Both programs run simultaneously, use the same referral code, and credit your account. The Affiliate Program provides significantly higher earning potential through multi-tier commissions, while the Referral Program offers a simpler view focused on direct referrals only.

Use the Affiliate Program dashboard for professional network building and serious income generation.
